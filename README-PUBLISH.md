# PromoWatch UA: публікація сайту

Це статичний сайт. Для публічного посилання потрібно завантажити вміст папки `promo-dashboard` на статичний хостинг.

Що має бути опубліковано:
- `index.html`
- `assets/grocery-dashboard-banner.png`
- `data/deals.json`

Папка `tools` потрібна тільки для оновлення даних і не обов'язкова для публічного сайту.

Найпростіший варіант:
1. Відкрити Netlify Drop або Cloudflare Pages.
2. Завантажити папку `promo-dashboard`.
3. Отримати публічне посилання.

Поточне локальне посилання `http://127.0.0.1:4173` працює тільки на цьому комп'ютері.
